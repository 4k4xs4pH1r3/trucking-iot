# Trucking Enrichment

A set of APIs for enriching data.  The different APIs that currently exist:


- DriverClassificationAPI
  - def isCerfied: Boolean
  - def wagePlan: String
- DriverTimesheetAPI
  - def hoursLogged: Int
  - def milesLogged: Int
- WeatherAPI
  - def isFoggy: Boolean
  - def isRainy: Boolean
  - def isWindy: Boolean
